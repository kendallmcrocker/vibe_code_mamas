# vibe_code_mamas

Go-to-market demo project.

## Setup

API credentials live in a local `.env` file (gitignored — never committed). Required keys:

| Variable | Source |
| --- | --- |
| `ANTHROPIC_API_KEY` | console.anthropic.com → API Keys |
| `OPENAI_API_KEY` | platform.openai.com → API keys |
| `POSTHOG_API_KEY` / `POSTHOG_HOST` | PostHog → Settings → Project |
| `META_APP_ID` / `META_APP_SECRET` / `META_AD_ACCOUNT_ID` / `META_ACCESS_TOKEN` | developers.facebook.com (Marketing API) |

Copy the variable names above into a `.env` file and fill in your own values.
